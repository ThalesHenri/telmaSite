Documentação do Sistema de Gerenciamento

1. Produto

Esta entidade permite registrar os produtos disponíveis. Cada produto contém os seguintes campos:

  • Nome: Nome do produto (campo obrigatório).
  • Preço: Valor do produto (campo obrigatório).
  • Peso: Peso do produto em gramas ou quilogramas (campo obrigatório).
  • Quantidade: Quantidade disponível do produto em estoque (campo obrigatório).
  • Foto: Imagem do produto para facilitar sua identificação visual (campo opcional).

Ações possíveis:

  • Adicionar novo produto.
  • Editar produto existente.
  • Excluir produto.
  • Visualizar lista de produtos cadastrados.

2. Cliente

Entidade responsável pelo cadastro dos clientes, contendo as seguintes informações:

  • Nome: Nome completo do cliente (campo obrigatório).
  • CPF: Cadastro de Pessoa Física, usado para identificação (campo obrigatório).
  • Telefone: Número de telefone para contato com o cliente (campo opcional).

Ações possíveis:

  • Adicionar novo cliente.
  • Editar dados de um cliente existente.
  • Excluir cliente.
  • Visualizar lista de clientes cadastrados.

3. Compras

A entidade compras é responsável por registrar todas as compras realizadas por clientes. Cada compra contém:

  • Nome do Cliente: Seleção de um cliente previamente cadastrado (campo obrigatório).
  • Produto Comprado: Seleção do produto que foi comprado (campo obrigatório).
  • Quantidade do Produto: Quantidade de unidades do produto compradas (campo obrigatório).
  • Preço: Preço final da compra, calculado de acordo com a quantidade e preço do produto (campo automático, mas pode ser ajustado manualmente).
  • Pagamento: Checkbox para marcar se o pagamento foi realizado ou está pendente.
  • Data da Compra: Data em que a compra foi realizada (campo obrigatório).

Ações possíveis:

  • Registrar nova compra.
  • Editar compra existente.
  • Excluir compra.
  • Visualizar todas as compras feitas.

Fluxo de Operação:

  1. Cadastro de Produto: Comece cadastrando todos os produtos disponíveis no sistema. Certifique-se de preencher corretamente o nome, preço, peso, quantidade, e, opcionalmente, adicionar uma imagem.
  2. Cadastro de Cliente: Antes de registrar uma compra, certifique-se de que o cliente esteja cadastrado no sistema. Insira os dados de identificação, como nome, CPF, e telefone.
  3. Registro de Compra: Ao efetuar uma venda, selecione o cliente e os produtos comprados. Registre a quantidade adquirida, marque se o pagamento foi realizado, e insira a data da compra.

Considerações Finais:

O sistema é intuitivo e simples, facilitando o gerenciamento de produtos, clientes e compras. Certifique-se de manter todos os registros atualizados para controle adequado de estoque e clientes.
